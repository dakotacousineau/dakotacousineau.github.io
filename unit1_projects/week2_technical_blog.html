"The differances between Borders, Margins, and Padding"

The main differances between Borders, Margins, and Padding are where they take place. Since all content in HTML is rectangle in shape think of them as boxes. The content is in a box. The margin is how far the box is from the edge of the screen. The border surrounds the box, and the padding is how far from the edge of the box the content will appear. Here is a handy image for a visual explaination.

Examples between Borders, Margins, and Padding are very similar as they affect the content in similar fashions. First you would need to indicate the class in the html for the paragraph. An example for a border would look something like this.
< p class="border">some text.< /p>
and the accompanying CSS would look like
p.border {
border-style:dotted;
border-width:2px;

}
as for the padding and margins just sub out the border for padding and margains and you also have options like top, bottom, left and right if you want seperate sides to have differant features.
