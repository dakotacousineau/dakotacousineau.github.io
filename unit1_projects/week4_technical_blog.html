"Week 4 array.map"

"Arrray.map"

Map is a method that invokes each element in an array and creates a new array containing that element. For example we have an array of array_name = [1, 2, 3] if call it with array_name.map { |x| x + 1 } it will run each object in the array with the command returning a new array with [2, 3, 4]. This is useful if you want to run a command on each object.
